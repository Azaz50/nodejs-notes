How Express.js Works (Very Short & Simple)

Express.js is a lightweight web framework for Node.js.

A client sends a request to the server.

Express processes the request through middleware (logging, auth, parsing).

It then matches the request to a route (like GET /users).

The route handler runs and sends a response back to the client.

âœ… How to Use Express.js (Step-by-Step, Very Simple)
1. Install Express
First, create a project and install Express:

npm init -y
npm install express

2. Create a Server File

Make a file named server.js (or app.js):

const express = require('express');
const app = express();

3. Create a Simple Route

This tells Express what to do when someone visits a URL:

app.get('/', (req, res) => {
  res.send('Hello Express!');
});

4. Start the Server

app.listen(3000, () => {
  console.log('Server running on port 3000');
});


Run it:

node server.js

When handling different operations
Example:

app.get('/tasks')       // Read tasks
app.post('/tasks')      // Create task
app.put('/tasks/:id')   // Update task
app.delete('/tasks/:id') // Delete task

# Pass Value through Route Parameters
http://localhost:3000/user/10

app.get('/user/:userid', (req, res) => {
    res.send(req.params) -------> { userid: '10' }
})

# Pass Value through Query Parameters
http://localhost:3000/search?name=yahubaba

app.get('/search', (req, res) => {
  res.send(req.query) -------> { name: 'yahubaba' }
})
